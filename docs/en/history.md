# History Tracking

> Review all practice sessions, identify areas for improvement through filtering and analysis.

## Overview
The History page provides a complete list of all your practice records, supporting multi-dimensional filtering and sorting, making it easy to find specific records to review or retry.

## Accessing the Page
- The **"All"** option in the Dashboard's quick filter

## Page Layout
The History page consists of the following areas:
- **Title Bar**: Includes Title "Practice Records", Filters button, Start Chat button
- **Filter Tags**: Displays the currently applied filter conditions
- **History Table**: Shows the complete history

---

## Filtering Features
Click the **"Filters"** button to open the filter panel. You can apply multiple filters simultaneously.

| Filter | Type | Options |
| --- | --- | --- |
| Age | Number Range | Min Age ~ Max Age (18-65) |
| Gender | Single Choice| Male / Female |
| Name | Text Input | By customer name |
| Income | Single Choice| 5M~10M / 10M~15M / 15M~20M / > 20M |
| Marital Status | Single Choice| Single / Married |
| Relationship | Single Choice| Family-Friends / Stranger |
| Stages Reached | Number Range | Min Stage ~ Max Stage |

---

## History Table

| Column | Description |
| --- | --- |
| Date | Date and Time of the practice |
| Age | Age of the persona |
| Gender | Gender of the persona |
| Name | Name of the persona |
| Occupation | Occupation of the persona |
| Income | Income level |
| Marital Status | Single / Married |
| Children | Number of children |
| Relationship | Family-friends / Stranger |
| Stage Reached | The final sales stage reached |
| Duration | Duration of the session |
| Actions | Action options |

### Action Buttons
- **View Review**: Navigate to the evaluation report to view detailed results.
- **Retry**: Start a completely new session using the exact same persona.
- **Continue**: Resume an unfinished session.

---

## Practice Tips

### Identify Weaknesses
Filter by "Stage Reached" to find sessions that only reached Stage 1 or 2. Review the Evaluation Report to identify common weaknesses and target them in future practices.

### Targeted Practice
Use filters (like Income, Marital Status) to find specific types of personas. Then, use the "Retry" feature to familiarize yourself with challenging scenarios.
