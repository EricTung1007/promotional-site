# Quick Start

> From login to completing your first practice session takes only 5 minutes.

## System Requirements

| Item | Requirement |
| --- | --- |
| Browser | Latest versions of Chrome, Edge, Firefox, or Safari |
| Network | Stable internet connection |
| Device | Desktop, Laptop, Tablet (Responsive Design Supported) |
| Microphone | Optional, required for voice input |

## Logging In

AI Coach uses enterprise SSO (Single Sign-On). No additional account registration is required.

### Login Methods

The platform supports the following SSO methods (depending on company settings):

1. **Sign in with Microsoft** — Click to authenticate via your corporate Microsoft account
2. **Sign in with Google** — Click to authenticate via your corporate Google account

> The login options displayed depend on your company's deployment environment. Some instances only provide Microsoft login.

### Login Flow

```text
Open Platform URL → Click SSO Login → Enter Corporate Credentials → Auto-redirect to Dashboard
```

### Troubleshooting Login Issues

| Status | Description | Solution |
| --- | --- | --- |
| Unregistered Account | Your account hasn't been added yet | Contact your admin to add your account |
| Disabled Account | "Account Disabled" prompt | Contact admin to check your account status |
| Auth Failed | "Authentication failed, please try again" | Refresh the page and retry, or clear cache |
| Too Many Requests | "Too many requests, please try again later" | Wait a few minutes before trying again |

## Your First Practice: Complete Flow

### Step 1: Login & View Dashboard
Log in to view the dashboard. All stats will be 0 on first use, which is normal.

### Step 2: Start Practice
Click the **"Start Conversation"** button on the dashboard to enter the practice page.

### Step 3: Select Scenario
You will see two scenario cards:

| Scenario | Description |
| --- | --- |
| Random Persona | System generates a random customer persona for practice. Best for quick starts. |
| Custom Persona | Define specific traits for targeted scenario training. |

> We recommend selecting **"Random Persona"** for your first try.

### Step 4: Chat Environment
After generation, you will enter the chat interface:
- **Left**: Customer profile sidebar displaying the auto-generated persona
- **Center**: Chat window to communicate with the AI
- **Top**: Timer indicating the duration of the current session

Type your message in the input box and press Enter to send.

### Step 5: End Practice
When ready to finish, click the **"End Conversation"** button in the top right.

### Step 6: View Review
The report takes 30 seconds to 2 minutes to generate. It includes:
- **Comprehensive Review** covering overall performance and stage completions
- **Line-by-line Feedback** for detailed evaluation and recommendations
