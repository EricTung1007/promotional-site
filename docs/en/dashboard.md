# Dashboard

> Keep track of your practice progress and learning outcomes at a glance

## Page Overview

The Dashboard is the home page after logging in, providing an overview of your personal learning progress and shortcuts. The page is divided into two areas: **Statistics Cards** and **Recent Practice History**, plus a top navigation bar for system controls.

## Top Navigation & System Controls
- **Logo/Home:** Returns you to the Dashboard from anywhere.
- **Language Switcher:** Toggle the interface language (Supports Traditional Chinese and Vietnamese).
- **AI Knowledge Assistant:** Opens a side panel with tools to support agents.
- **Admin Panel:** For authorized users to configure platform settings.
- **Logout:** Exits the current session.

## Statistics Cards 

### Start Conversation
- A prominent button to enter the practice setup flow and begin a new session.

### Cumulative Practice Count
- Total number of practice sessions you have initiated.
- Unit: **Times**

### Total Conversation Time
- Cumulative duration of all your active practice sessions.
- Unit: **Hours**

### Consecutive Practice Days
- Number of uninterrupted days you have practiced.
- Unit: **Days** (Resets if a day is missed)

## Recent Practice History
Displays your 5 most recent practices in a detailed data table.

### Quick Filters

| Option | Description |
| --- | --- |
| Clock Icon (Default) | Displays the 5 most recent practice records |
| Today | Filters practice records for today |
| Date Picker | Filter practice records for a specific date (YYYY-MM-DD)|
| All | Jumps to the full "History" page |

### Data Columns
| Field | Description |
| --- | --- |
| Date | Date the practice was conducted |
| Age | Age of the customer persona |
| Gender | Gender of the customer persona |
| Name | Name of the customer persona (Clickable to view detailed profile) |
| Occupation | Occupation of the customer persona |
| Income | Income level of the customer persona |
| Achieved Stage | The highest sales phase reached (Includes a hover tooltip for details) |
| Duration | Duration of the conversation (Minutes) |

### Action Buttons

| Button | Description |
| --- | --- |
| Review (Eye Icon) | Go to the detailed Evaluation Report page |
| Redo (Recycle Icon) | Start a completely new session using the exact same persona |
| Continue (Play Icon) | Resume an unfinished conversation |

### Persona Profile Modal
Clicking on the customer's **Name** inside the table opens a detailed profile including:
- **MBTI Personality Type:** Helps you understand their decision-making style.
- **Family & Education Details:** Marital status, education level.
- **Relationship:** The relationship context between the agent and the customer.
- **"Redo with this role" Button:** Instantly starts a flow with this exact profile.
