# Dashboard

> Keep track of your practice progress and learning outcomes at a glance

## Page Overview

The Dashboard is the home page after logging in, providing an overview of your personal learning progress and shortcuts. The page is divided into two areas: **Statistics Cards** and **Recent Practice History**.

## Statistics Cards 

### Start Conversation
- Click to enter the practice page and start a new session

### Total Practices
- Total number of practice sessions you have completed
- Unit: **Times**

### Total Conversation Time
- Cumulative duration of all your practice sessions
- Unit: **Hours**

### Consecutive Practice Days
- Number of consecutive days you have practiced
- Unit: **Days** (Resets if a day is missed)

## Quick Filters

| Option | Description |
| --- | --- |
| Clock Icon (Default) | Displays the 5 most recent practice records |
| Today | Filters practice records for today |
| All | Jumps to the "History Tracking" page |

## Recent Practice History

| Field | Description |
| --- | --- |
| Date | Date the practice was conducted |
| Age | Age of the customer persona |
| Gender | Gender of the customer persona |
| Name | Name of the customer persona |
| Occupation | Occupation of the customer persona |
| Income | Income level of the customer persona |
| Stage | Final sales stage reached |
| Duration | Duration of the conversation (minutes) |

### Action Buttons

| Button | Description |
| --- | --- |
| View Review | Go to the evaluation report page |
| Retry | Start a new session using the same persona |
| Continue | Resume an unfinished conversation |

### Date Grouping
Records are grouped by date if there are multiple sessions on the same day.
