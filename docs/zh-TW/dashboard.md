# 儀表板

> 一目了然掌握你的練習進度與學習成果

## 頁面概覽

儀表板是登入後的首頁，提供個人學習概況與快速操作入口。頁面由上而下分為兩個區域：**統計卡片區** 與 **近期練習紀錄**。同時，上方導覽列提供了核心系統操作功能。

## 頂部導覽列與系統控制
- **Logo/首頁：** 從任何位置返回儀表板。
- **語系切換：** 切換介面語言（支援繁體中文與越南語）。
- **AI 知識助手：** 開啟側邊面板，提供支援業務人員的各種工具與範例。
- **後台管理系統：** 供具權限之管理員進行系統與提示詞設定。
- **登出：** 登出目前的系統會話。

## 統計卡片區

### 開始對話
- 點擊即可進入對話練習設定頁面，開始新的模擬演練。

### 累積練習次數
- 顯示您截至目前為止展開的所有練習總次數。
- 單位：**次**

### 對話總時間
- 顯示您所有練習的累計對話時長。
- 單位：**小時/分鐘**

### 連續練習的天數
- 顯示您連續每日進行練習不中斷的天數紀錄。
- 單位：**天**（只要有一天未啟動練習，紀錄就會歸零重新計算）

## 近期練習紀錄

顯示最近的練習紀錄列表（預設為 5 筆），並提供詳細的資料與快速操作。

### 快速篩選

| 選項 | 說明 |
| --- | --- |
| 時鐘圖示（預設） | 顯示最近 5 筆練習紀錄 |
| 今天 | 篩選出今日的練習紀錄 |
| 日期選擇器 | 依據指定的日期過濾紀錄 (YYYY-MM-DD)|
| 全部 | 跳轉至「練習歷史」頁面，查看完整紀錄 |

### 資料欄位

| 欄位 | 說明 |
| --- | --- |
| 日期 | 練習進行的日期 |
| 年齡 | 客戶角色的年齡 |
| 性別 | 客戶角色的性別（男 / 女） |
| 姓名 | 客戶角色的姓名（可點擊查看詳細人物誌） |
| 職業 | 客戶角色的職業 |
| 收入 | 客戶角色的收入等級 |
| 達成階段 | 該次練習最終達成的最高銷售階段（附帶工具提示說明） |
| 練習時長 | 該次練習的實際對話時長 (分鐘)|

### 操作按鈕

| 按鈕 | 說明 |
| --- | --- |
| 查看評估 (眼睛圖示) | 前往該次練習的詳細評核報告頁面 |
| 重新挑戰 (循環圖示) | 使用完全相同的客戶角色設定，開始一次全新的練習 |
| 繼續練習 (播放圖示) | 接續上次未完成的對話 |

### 人物誌檔案面板 (Persona Modal)
在列表中點擊客戶的 **姓名**，將會開啟一個詳細的人物檔案對話框，包含：
- **MBTI 人格特質：** 協助您了解客戶的決策與溝通風格（例如：INTJ - 建築師）。
- **家庭與教育資訊：** 婚姻狀態、學歷等背景。
- **關係設定：** 客戶與業務員當前的熟稔程度。
- **「使用此角色重新練習」按鈕：** 直接以該詳細設定啟動新的模擬練習。
