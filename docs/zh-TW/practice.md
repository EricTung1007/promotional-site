# 對話練習

> 與 AI 客戶進行擬真銷售對話，磨練你的溝通技巧

## 概覽

對話練習是 AI Coach 的核心功能。您將與由 AI 驅動的虛擬客戶進行模擬銷售對話，客戶會根據其設定的屬性（年齡、職業、收入、人格特質等）做出擬真反應，讓您在安全的環境中練習各種銷售情境。

## 進入練習

從以下入口進入對話練習頁面：

- 儀表板的 **「開始對話」** 按鈕
- 練習歷史的 **「重新挑戰」** 或 **「繼續練習」** 按鈕

---

## 場景選擇

進入練習頁面後，首先選擇練習場景。系統提供以下選項：

### 隨機角色

- **說明**：系統將為您隨機選擇一個角色進行對話練習
- **適用情境**：想快速開始練習、不需特定客戶類型時
- **行為**：點擊後直接開始產生角色，無需額外設定

### 自訂角色

- **說明**：您可以自行定義角色以進行對話練習
- **適用情境**：想針對特定客戶類型訓練（例如高收入已婚客戶）
- **行為**：點擊後開啟「客戶樣態設定」面板

### 重新對練（來自歷史紀錄）

- **說明**：使用與之前練習相同的客戶角色，開啟全新對話
- **適用情境**：想對同一位客戶嘗試不同策略
- **入口**：僅透過儀表板或練習歷史的「重新挑戰」按鈕觸發

### 繼續對練（來自歷史紀錄）

- **說明**：接續之前未完成的對話
- **適用情境**：上次練習中途離開，想繼續完成
- **入口**：僅透過儀表板或練習歷史的「繼續練習」按鈕觸發
- **特別說明**：客戶對上次對話有模糊印象，您可以藉由上次對話的內容繼續互動

## 客戶樣態設定

選擇「自訂角色」後，會開啟客戶樣態設定面板。所有欄位皆為選填，未設定的項目會由 AI 自動補充。

### 基本資料

| 欄位 | 類型 | 選項 / 範圍 | 說明 |
| --- | --- | --- | --- |
| 年齡 | 下拉選單 | 18 ~ 65 歲 | 客戶年齡 |
| 性別 | 下拉選單 | 男 / 女 | 客戶性別 |
| 個性 (MBTI) | 下拉選單 | 16 種 MBTI 類型 | 影響客戶的溝通風格與反應模式 |

| 職業 | 搜尋選單 | 100+ 種職業選項 | 支援關鍵字搜尋，輸入後即時過濾 |

### 家庭狀況

| 欄位 | 類型 | 選項 / 範圍 | 說明 |
| --- | --- | --- | --- |
| 婚姻狀況 | 下拉選單 | 單身 / 已婚 | 影響客戶的保險需求面向 |
| 配偶年齡 | 下拉選單 | 18 ~ 65 歲 | 僅在「已婚」時可選擇 |
| 子女數量 | 下拉選單 | 0 ~ 10 人 | 設定後可進一步設定每位子女的性別與年齡 |

當子女數量 > 0 時，會動態展開每位子女的設定欄位：

| 欄位 | 類型 | 說明 |
| --- | --- | --- |
| 子女 N 性別 | 下拉選單 | 男 / 女 |
| 子女 N 年齡 | 下拉選單 | 最大年齡 = 客戶年齡 - 18 |

### 與業務員關係

| 欄位 | 類型 | 說明 |
| --- | --- | --- |
| 是否為陌生客戶？ | 開關切換 | 預設為「是」（陌生客）。關閉後切換為「親友」模式 |

當設定為 **親友** 時，會展開以下額外欄位：

| 欄位 | 類型 | 說明 |
| --- | --- | --- |
| 我對客戶稱謂 | 文字輸入 | 您如何稱呼客戶？例如：小王、王大哥 |
| 認識時間 | 文字輸入 | 從小到大、3年、5年、10年... |
| 與業務員關係描述 | 文字區域（必填） | 描述您與客戶的關係背景 |

#### 關係描述範本

系統提供 5 個預設範本供快速套用：

1. **舊客戶** — 長期保險客戶的關係描述
2. **親戚** — 家族親戚關係的描述
3. **前同事** — 前公司同事的關係描述
4. **同學** — 學校同學的關係描述
5. **陌生訪談** — 街頭訪問式的關係描述

### 操作按鈕

| 按鈕 | 說明 |
| --- | --- |
| 清除所有 | 重設所有欄位為初始值 |
| 記錄 | 開啟歷史設定紀錄列表，可載入之前使用過的角色設定。最多可儲存一定筆數的紀錄 |
| 取消 | 關閉設定面板，不進行任何操作 |
| 確認 | 使用目前設定開始練習 |

### 輸入驗證

- 所有文字輸入欄位會進行安全性檢查（防止 XSS / SQL 注入）
- 親友模式下「關係描述」為必填欄位
- 不安全字元會顯示警告：「請勿輸入特殊字元」

## 角色生成

確認設定後（或選擇隨機角色後），系統會進入角色生成階段：

- 畫面顯示載入動畫與場景資訊
- 提示訊息：「提示：生成需要一些時間，請耐心等候。」
- 生成完成後自動進入對話介面

生成的客戶角色包含完整的個人資料，即使您只設定了部分屬性，AI 也會自動補齊其餘欄位。

## 對話介面

### 版面配置

| 區域 | 位置 | 說明 |
| --- | --- | --- |
| 客戶資訊側邊欄 | 左側 | 顯示客戶的完整角色設定資料，可收合 |
| 場景說明 | 對話區上方 | 說明本次對話的情境背景 |
| 對話訊息列表 | 中央 | 您與 AI 客戶的對話紀錄，自動捲動至最新訊息 |
| 訊息輸入區 | 下方 | 文字輸入框與傳送按鈕 |
| 計時器 | 右上方 | 顯示目前練習時長（MM:SS 格式） |
| 結束對話按鈕 | 計時器旁 | 點擊可結束本次練習 |

### 場景說明

根據不同的練習模式，場景說明會有所不同：

| 模式 | 場景說明 |
| --- | --- |
| 新對話（陌生客） | 「您是第一次與 {客戶名} 對話」 |
| 新對話（親友） | 「您與 {客戶名} 互相認識，但電話並沒有顯示您的個人資訊」 |
| 繼續對練 | 「您與 {客戶名} 不久前通過電話。他對上次對話有些模糊的印象，您可以藉由上次對話的內容，繼續與他對話。」 |

所有模式都會提醒：「請先查閱左方的客戶資料，了解客戶的保險需求，再進行聊天。」

### 發送訊息

1. 在輸入框中輸入文字（預設提示文字：「傳送訊息至 PASSION Agent...」）
2. 按 **Enter** 或點擊 **傳送按鈕** 發送
3. 等待 AI 客戶回應（回應過程中輸入框暫時停用）
4. AI 回應完成後即可繼續對話

> 輸入區下方提示：「PASSION Agent 可能會產生不準確的訊息。請驗證重要數據並遵循適用的法規和政策。」

### 內容安全檢測

系統內建內容安全機制。當您的訊息觸發安全檢測時，訊息不會被送出，並顯示對應的提示訊息：

| 檢測類型 | 提示訊息 |
| --- | --- |
| 越權操作/繞過安全 | 系統偵測到疑似越權操作或嘗試繞過安全限制，請避免輸入可疑內容 |
| 不當或高風險內容 | 系統偵測到疑似不當或高風險內容，請避免輸入具爭議、危險或不適當的訊息 |
| 非法或高敏感內容 | 系統偵測到疑似非法或高敏感內容，請勿輸入違法資訊 |
| 個資或敏感資料 | 系統偵測到疑似個資或敏感資料（如電話、地址、身分證號等），請避免輸入個人資訊 |
| 惡意連結 | 系統偵測到疑似危險或惡意連結，請避免輸入不安全的網址 |

## 銷售階段

AI Coach 的練習圍繞多個銷售階段進行。各階段的定義與數量由管理員設定，常見的階段例如：

| 階段 | 目標 | 說明 |
| --- | --- | --- |
| 第 1 階段 | 破冰/建立信任 | 與客戶建立初步信任關係，了解基本狀況 |
| 第 2 階段 | 需求探索 | 深入了解客戶的保險需求與擔憂 |
| 第 3 階段 | 方案說明 | 針對客戶需求提供合適的保險方案 |
| 第 4 階段 | 異議處理 | 回應客戶的疑慮與反對意見 |
| 第 5 階段 | 促成 | 引導客戶做出購買決定 |

> 實際階段名稱與數量依公司管理員設定為準。

### 階段判定方式

階段**不會在對話過程中即時顯示**。對話結束後，AI 會回顧完整對話內容，分析您在每個階段的表現，判定您最終達成了哪個階段。結果會呈現在評量報告的「階段評量」區塊中。

換句話說，對話過程中請專注於與客戶的互動，不需擔心「目前在第幾階段」。階段評量是事後回顧式的分析，不影響對話進行。

## 結束對話

### 主動結束

1. 點擊右上方的 **「結束對話」** 按鈕
2. 系統彈出確認對話框
3. 確認後，系統會自動結束對話並開始產生評量報告
4. 頁面跳轉至評量報告頁面

> 注意：AI 回應進行中時，「結束對話」按鈕暫時停用。

### 客戶發起結束

當 AI 客戶在對話中表達想要結束時，頁面頂部會出現通知列：

- 標題：「客戶想結束對話」
- 描述：「您要繼續對話還是結束本次對話？」
- 兩個選項：
  - **繼續對話** — 忽略客戶的結束意圖，嘗試繼續溝通
  - **結束對話** — 同意結束，進入評量產生流程

### 結束後的處理

對話結束後，系統會在背景自動進行以下處理：

1. **儲存對話紀錄** — 記錄練習時長與所有對話內容
2. **產生階段評量** — AI 分析您在各銷售階段的表現
3. **產生綜合評核** — AI 撰寫完整的表現評估報告
4. **初始化回饋評分** — 建立每則訊息的評分記錄（預設中立）
